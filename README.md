# 🤖 AI Face-Off: Debate Platform

## 📌 Overview
AI Face-Off is an interactive web application where two AI models debate a given topic while a third AI model evaluates the outcome.

- One AI supports the topic  
- Another AI opposes it  
- A judge AI evaluates and declares a winner

## ▶️ How to Use
1. Click **Start**  
2. Select a **supporter AI model** (Gemini or Cohere)  
3. Enter API keys for:
   - Supporter AI  
   - Opposer AI  
   - Judge AI (OpenAI)  
4. Enter a **debate topic**  
5. Click **Start** to generate arguments  
6. Click **Evaluate** to see the winner  

---

## 🚀 Features
- AI vs AI debate (Gemini vs Cohere)  
- Automated pro & counter arguments  
- AI-based evaluation with scoring criteria  
- Real-time event tracking  
- Simple interactive UI
